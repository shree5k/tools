Generate product aligned copy instantly

How to use : Default
  1. Install Ollama from ollama.com. Run `ollama run llama3.2` in your terminal. (Or add your Claude/OpenAI API key)
  2. Each column is a UI context. Button, Tray Title, Error Message, you name it
  3. Drop your text into any cell
  4. Press Tab Tab to generate 10 variations. Cmd/Ctrl+Enter for one
  5. Drag the blue dot to generate variations in any range
  6. Use arrow keys to navigate, Enter to edit, Delete to clear
